# TradePal AI - Frontend

Next.js chat interface for TradePal AI customer service system.

## Quick Start

```bash
npm install
npm run dev
```
Open http://localhost:3000

## Features (Step 1 ✅)

- Chat interface with message bubbles
- State management for conversation history
- Loading states and error handling
- Responsive design with auto-scroll
- Timestamps and modern UI
- Mock responses (backend in Step 2)

## Components

- `ChatContainer` - Main container with state
- `MessageList` - Message display
- `ChatInput` - Input with send button

## Tech Stack

Next.js 16 • TypeScript • Tailwind • shadcn/ui
